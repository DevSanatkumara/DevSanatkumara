Аудиоплеер

Аудиоплеер используется в следующих разделах:
  • Аудиолекции
  • Священные тексты
  • Книги и упадеши Гуру
  • Песни пробужденного
  • Бхаджаны
  • Музыка для практики (если будет такой раздел)

Требования к плееру

Требования:
  • Единый вид для всех вариантов использования на сайте
  • "Бесшовное" проигрывание (трек не останавливается при переходе на другую страницу)
  • Расположение: расположен внизу страницы, закреплен при скроллировании

Вид плеера

Дизайнеру:
  1. Можно увеличить длину и перераспределить кнопки на свободном пространстве. Учитывать разрешения экранов.
  2. Можно часть кнопок спрятать во всплывающее меню, открывающееся по кнопке [...] (скорость, зацикливание, добавление в очередь и в плейлист).
  3. Кнопки быстрой перемотки можно сделать так .
  4. Можно еще добавить кнопку "Поделиться" в конец.

Описание функциональных элементов

  1. Заголовок — подгружается из поля "Название":
    • для лекций — это название лекции;
    • для аудиокниг — название книги;
    • для Песен пробужденного — название песни;
    • для бхаджанов — название бхаджана.
  2. Название главы — подгружается справа от заголовка, после точки и пробела, из названия главы, добавленной в элемент книги:
    • для лекций — если лекция состоит из нескольких частей, выводится Название части, например: Лекция 23.07.2023. Часть 1.
  3. Автор — подгружается из поля "Автор" в элементе книги/лекции.
    • Для бхаджанов можно подтягивать год исполнения или название группы бхаджанов (для этого нужно создать перечень в элементе бхаджана, например: "Бхаджаны господу Вишну").
  4. Чтец — выводится из поля "Читает" для аудиокниг.
  5. Ползунок воспроизведения — окрашивается по мере воспроизведения аудио, при нажатии на любое место переходит на это место в аудио. Для авторизированных пользователей: при достижении конца аудио оно отмечается как "прослушанное".
  6. Время воспроизведения — чч:мм:сс — слева прямой отсчет, справа — обратный.
  7. Настройка громкости — по умолчанию на середине.
  8. Настройка скорости воспроизведения — при нажатии открывает всплывающее меню вида:
    • или
      (можно или ползунок, или кнопки).
  1. Кнопка зацикливания — если включена, выделяется цветом.
  2. Кнопка плей/пауза.
  3. Кнопки перемотки — в начало и конец трека.
Кнопки быстрой перемотки

Нужно определиться со временем — можно 15 сек назад и 20 вперед?

Дополнительные функции

  1. Кнопка добавления в очередь воспроизведения — кнопка открывает всплывающее меню с командами:
    • Добавить в очередь
    • Открыть очередь
      Если очереди еще нет, вторая команда неактивна. Если есть, то открывается окно с названием "Очередь воспроизведения".
  • трек добавляется в конец очереди.

Опции окна:
  • Кнопка слева — переместить трек перетаскиванием;
  • Кнопка справа — открывает встроенное меню с командами "Удалить", "Поделиться", "Добавить в плейлист" (для неавторизированного пользователя при нажатии выводить сообщение "Доступно после авторизации", описание добавления в плейлист см. ниже);
  • Верхняя и нижняя строка в названии трека:
    • для лекций — [Название] и [дата лекции];
    • для книг — [Название] и [автор];
    • для бхаджанов — [Название] и [группа бхаджанов];
  • Продолжительность трека чч:мм:сс.
  • Запустить воспроизведение по нажатию на трек, трек окрашивается цветом.
  • Над треками выводится [порядковый номер играющего трека] из [кол-во треков], иконка таймера, сумма времени всех треков.
  • Закрытие окна по крестику в правом углу.
    Для авторизированного пользователя Очередь хранится в ЛК.
  1. Кнопка добавления в плейлист — для неавторизированного пользователя при нажатии выводить сообщение "Доступно после авторизации", для авторизированного...

Исходные требования

Функции плеера:
  • Стандартные:
    • перемотка,
    • play/pause,
    • следующий, предыдущий,
    • зациклить,
    • shuffle,
    • громкость,
    • переход на 10-15 секунд вперед и назад,
    • скорость.
  • Плейлист:
    • Добавить в плейлист.
    • Создать новый и добавить.
    • Добавить в очередь.
  • Добавить в очередь прослушивания
  • показать очередь прослушивания
    • с возможностью
    • менять местами лекции
    • удалять из списка
  • По клику на время можно поменять с общего время на то, сколько еще осталось до конца.
